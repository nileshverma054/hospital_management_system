# Hospital Management System
A RESTful API for a Hospital Management System using Python Flask framework that allows users to manage patients, doctors, and departments efficiently.


### Tech Stack
- Python 3.12.1
- Flask 3.0.3
- Flask-RESTful
- SQLAlchemy
- MySQL

## Run App
```
bash startup.sh
```
